## Red Flags to Avoid

### DON'T DO:
- [ ] Don't use auto-generated or AI-only content (update with human review)
- [ ] Don't buy backlinks (white hat only)
- [ ] Don't keyword stuff (natural language only)
- [ ] Don't hide links or text
- [ ] Don't duplicate content across sites
- [ ] Don't ignore Google Search Console errors
- [ ] Don't neglect mobile users

### DO:
- [ ] Do focus on user intent first
- [ ] Do write for humans, not robots
- [ ] Do build links naturally through great content
- [ ] Do respond to customer reviews
- [ ] Do update old content regularly
- [ ] Do monitor Core Web Vitals
- [ ] Do test on real devices
